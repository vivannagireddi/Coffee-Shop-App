# Coffee Shop App
This App is a non-proprietary Service App designed as a prototype to test the connectivity between the Webpage Interface(HTML + CSS + JavaScript) <br/>
and the Backend Interface(Python + MySQL). 


## How to Use
There are two files: 
1. <code>WebApp.py</code> - If you want to initiate a flask server to run in your browser.
2. <code>DesktopApp.py</code> - If you want to run the Application as a Desktop App using the <code>CEF</code> Renderer (Supports python versions upto 3.9.x). It's obsolete and doesn't support much web component capabilities, like blur, etc. It's recommended not to use it as it contains a lot of bugs.

## Note
The development of this App has been terminated. This App will not be maintained in the future, as it is a school project. Also, this App may contain bugs which will not be fixed.
